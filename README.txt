マネーサウンド:OFX/CSV/PDF出力対応のアカウントアグリゲーションサービス

マネーサウンドは、対応金融機関の口座情報一覧を取得するアカウントアグリゲーションサービスです。取得した明細データは、OFX/CSV対応ソフト・サービスに取り込めるほか、PDFでの保存や、口座一覧の印刷にも対応しています。

マネーサウンドを実行するには、PHPバージョン5.4以降が必要です。
